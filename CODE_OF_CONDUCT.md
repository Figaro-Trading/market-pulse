# Code of Conduct

This project adopts the **Contributor Covenant v2.1** as its code of
conduct. The full text is published at:

https://www.contributor-covenant.org/version/2/1/code_of_conduct/

By participating in this project (commits, issues, pull requests,
discussions, etc.), you agree to follow this code.

## Scope

This code applies to all project spaces — the GitHub repository, issue
tracker, pull requests, discussions, and any other forum used for
project-related communication.

## Reporting

To report a violation, contact the maintainer via a private channel:

- Open a [GitHub Security Advisory](https://github.com/Figaro-Trading/market-pulse/security/advisories/new)
  (this channel is private by default and not visible to other users).

Reports will be reviewed and acted on within 7 days. All reporters will
be kept confidential.

## Enforcement

The maintainer is responsible for clarifying and enforcing this code.
Actions may include warnings, temporary bans, or permanent removal from
the project's communication channels, in line with the guidelines from
the Contributor Covenant v2.1.

---

For the full text of the rules and enforcement guidelines, please see
the official Contributor Covenant page linked above.
